# Ecommerce Customers - Linear Regression Model 📊

### Project Overview
This project uses a **Linear Regression** model to analyze customer data for an Ecommerce company. The goal is to decide whether the company should focus its efforts on their **Mobile App** experience or **Website**.

### Dataset Features:
* **Avg. Session Length**: Average session of in-store style advice sessions.
* **Time on App**: Average time spent on App in minutes.
* **Time on Website**: Average time spent on Website in minutes.
* **Length of Membership**: How many years the customer has been a member.
* **Yearly Amount Spent**: The target variable (Amount spent annually).

### Key Findings:
* **Length of Membership** is the most significant factor (Coeff: ~61.27).
* **Time on App** is much more influential than **Time on Website**.
* **Model Performance**: RMSE is approximately 8.93, indicating high accuracy.

### Technologies Used:
* Python 3
* Pandas & Numpy
* Scikit-Learn
* Matplotlib & Seaborn
